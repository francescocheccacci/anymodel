# AnyModel

## Live Website
- [AnyModel](https://anymodel.pythonanywhere.com/)

### Development Requirements
- Python 3.6


## Setup
### Local

1. Clone the repo:
```bash
git clone https://github.com/francescocheccacci/anymodel.git
```

1. Install packages:
```bash
pip install -r requirements.txt
```

3. Create db:
```bash
flask db upgrade
```
4. Launch the project:
```bash
python app.py
```
